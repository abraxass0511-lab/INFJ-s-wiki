```bash

```

## 📋 트리거 1 실행 절차 — "위키야 정리해줘"

### 1단계: 학습 내용 추출

현재 대화를 분석하여 아래 항목을 추출. **잡담과 디버깅 과정은 제외**, 핵심만 뽑는다.

| 필드 | 필수 | 설명 | 예시 |
|------|------|------|------|
| `title` | ✅ | 학습 주제 | `"ACE-Step 음악 생성 BPM 최적화"` |
| `summary` | ✅ | 한 줄 핵심 요약 | `"145-160 BPM이 최적 리텐션 구간"` |
| `agentName` | ✅ | 에이전트 이름 | `"루나"`, `"알파"`, `"레오"` 등 |
| `agentDomain` | ✅ | 에이전트 도메인 | `"YouTube 음악 채널"` 등 |
| `keyLearnings` | ✅ | 핵심 배운 점 (최소 1개) | `["BPM 145가 리텐션 최적"]` |
| `decisions` | 선택 | 결정한 사항들 | `["1일 3영상 스케줄 확정"]` |
| `skills` | 선택 | 새로 익힌 스킬/패턴 | `["FFmpeg 비트싱크"]` |
| `connections` | 선택 | 관련 기존 지식 키워드 | `["리텐션 전략"]` |

### 2단계: `00_Raw/`에 마크다운 저장

```
C:\Users\YS\Desktop\안티그래피티\위키에이전트\00_Raw\YYYY-MM-DD_에이전트명_제목.md
```

**파일명 예시:**
- `2026-04-22_루나_BPM_최적화_전략.md`
- `2026-04-22_알파_반도체_섹터_분석.md`

**마크다운 포맷:**

```markdown
# {title}

> 📅 {YYYY-MM-DD} {HH:MM} | {agentName} 에이전트 대화에서 자동 추출
> 🤖 에이전트: {agentName} ({agentDomain})
> 🔗 Conversation: {conversationId}

## 💡 핵심 요약
{summary}

## 📖 오늘 배운 것
- {keyLearnings[0]}
- {keyLearnings[1]}

## ⚖️ 결정한 사항
- {decisions[0]}

## 🚀 새로운 스킬/패턴
- {skills[0]}

## 🔗 연관 지식
- [[{connections[0]}]]
- [[{connections[1]}]]

---
*Source: {agentName} Agent Conversation Export*
```

### 3단계: 저장 완료 보고

사용자에게 보고:
- ✅ 파일명
- 📂 저장 경로
- ⏳ "구조화해" 라고 하시면 위키로 분류합니다

> ⚠️ 이 단계에서는 **저장만**. `npm run process`는 실행하지 않는다.

## 📋 트리거 2 실행 절차 — "구조화해"

### 1단계: `npm run process` 실행

```bash
cd "C:\Users\YS\Desktop\안티그래피티\위키에이전트"
npm run process
```

이 명령이 자동으로:
1. `00_Raw/`의 모든 `.md/.txt` 파일을 RL 기반 분류
2. `10_Wiki/{카테고리}/` 에 Karpathy 위키 템플릿으로 생성
3. `20_Meta/Graph.json` 에 노드/엣지 추가
4. `20_Meta/Policy.md` 에 보상 기록
5. Git 커밋 + GitHub Push

### 2단계: 결과 검증 및 보고

- ✅ 각 파일이 어떤 카테고리에 배치되었는지
- 🔗 몇 개의 기존 지식과 연결되었는지
- 📦 Git 커밋 해시

> ⚠️ **절대 룰**: 실행 후 출력만 믿지 말고, 실제 파일이 `10_Wiki/` 하위에 생성되었는지 독립 검증할 것.

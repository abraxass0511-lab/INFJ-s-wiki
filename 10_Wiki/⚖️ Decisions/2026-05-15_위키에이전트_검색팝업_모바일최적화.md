### 세부 내용
- 기존 `<select>` 드롭다운 → 체크박스 팝업으로 교체하여 복수 연도 동시 검색 가능
- `전체` 체크박스 추가: 모든 연도 일괄 체크/해제 토글
- 버튼 라벨 동적 표시: `2026년 ▾` / `2026년 외 1개 ▾` / `전체 ▾`
- 적용 버튼 클릭 시 팝업 닫힘 + 검색 재실행
- `populateYearCheckboxes()` — 연도 목록 + 전체 체크박스 동적 생성
- `toggleAllYears(checked)` — 전체 체크/해제 시 모든 연도 동기화
- `syncAllCheckbox()` — 모든 연도 체크 시 전체 체크박스 자동 체크
- `getSelectedYears()` — `[value]` 속성 필터로 전체 체크박스(`"on"` 값) 제외
- `updateYearBtnLabel()` — 선택 상태에 따른 버튼 라벨 동적 변경
- `applyYearFilter()` — 적용 버튼 클릭 시 팝업 닫기 + doSearch 재실행
- `<input type="checkbox">` 에 `value` 속성을 안 넣으면 기본값이 `"on"` — `querySelectorAll('[value]:checked')` 로 필터링 필수
- **모바일 문제**: `aspect-ratio:1` + `overflow:hidden` → 카테고리 태그 3개 이상이면 잘림
- **해결**: `aspect-ratio:1` 제거, `overflow:hidden` 제거, `min-height:80px` 적용
- **원칙**: 동적 콘텐츠(태그 개수가 변함)에는 고정 비율 대신 `min-height` 사용
- 팝업 방식 채택 (드롭다운 대비 복수 선택 UX 우수)
- 전체 체크박스를 기본 포함 (연도가 많아질 것에 대비)
- `min-height` 기반 가변 높이로 모바일 셀 설계 (태그 가독성 우선)

### 세부 내용
- 기존에는 `<select>` 드롭다운으로 연도를 선택했음 (단일 선택만 가능)
- 데이터가 여러 해에 걸쳐 쌓이면 **복수 연도를 동시에 검색**할 필요가 있음
- **`populateYearCheckboxes()`**: 연도 목록 + 전체 체크박스 동적 생성
- **`toggleAllYears(checked)`**: 전체 체크/해제 시 모든 연도 동기화
- **`syncAllCheckbox()`**: 모든 연도가 체크되면 전체도 자동 체크
- **`getSelectedYears()`**: `[value]` 속성 필터로 전체 체크박스 제외
- **`applyYearFilter()`**: 적용 버튼 클릭 시 검색 재실행
- 모바일에서 날짜 셀에 카테고리가 여러 개 있을 때 일부만 보이고 나머지가 잘림
- `aspect-ratio:1` (정사각형 고정) + `overflow:hidden` → `min-height:80px`으로 변경
- 셀 높이가 내용에 맞게 자동 확장되어 모든 태그 표시
- HTML 체크박스에 `value` 속성이 없으면 기본값은 `"on"` → 필터링 필수
- 동적 콘텐츠에는 `aspect-ratio` 대신 `min-height` 사용이 적합

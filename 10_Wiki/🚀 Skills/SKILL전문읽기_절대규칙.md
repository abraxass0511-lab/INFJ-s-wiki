# [[SKILL.md 전문 읽기 절대 규칙]]
## 📌 한 줄 통찰
> 에이전트가 SKILL.md를 grep 부분 검색만 하고 전문을 읽지 않아, 모든 규칙(황금 시간대, 분 jitter, 제목 규칙, 장르 매핑)을 빼먹은 사건 발생. 전문 읽기를 최상위 절대 규칙으로 등록함.
## 📖 구조화된 지식
- **사건:** 영상 제작 시 SKILL.md의 슬롯 스케줄(KST 21:00/01:00/06:00)을 무시하고 임의 시간 제안, 분 jitter 규칙 누락 발생.
- **근본 원인:** 세션 시작 시 전문 읽기 없이 grep 검색만으로 규칙 확인 완료 판단.
- **절대 규칙 (SKILL.md Section 0 최상단 등록):**
    1. 세션 시작 시 SKILL.md를 view_file로 1행~마지막행 전문 읽기 (800줄 이하이므로 1회 호출).
    2. 작업 전 체크리스트 자기검증 (슬롯/jitter/제목/장르/음악이미지독립/WAV변환/Python분리).
    3. grep 부분 검색만으로 규칙 확인 완료 판단하는 것 금지.
    4. KNOWLEDGE_BASE.md, prompt_templates.json도 함께 읽기.

### 세부 내용
- **API 사용 비용 문제:** 외부 AI 모델의 API를 많이 사용할 경우 발생하는 높은 비용 문제.
- **AI 모델 선택의 어려움:** AI 개발 프로젝트의 목적(수익성, 가성비 등)에 따라 어떤 인공지능 모델을 선택해야 할지에 대한 고민.
- *   **문제점:** 다른 사람이 API를 많이 사용하면 비용이 많이 발생하고, 데이터 프라이버시 및 통제권 확보가 어려움.
- *   **해결/배운점:** 로컬 LLM(Large Language Model)을 직접 구축하여 사용해야 함.
- *   **주요 장점:**
- *   **완벽한 프라이버시:** 데이터를 외부로 전송할 필요 없이 자체 환경에서 처리하여 보안 강화.
- *   **오프라인 코딩:** 인터넷 연결 없이도 개발 및 테스트 가능.
- *   **무제한 무료 사용:** API 사용료 없이 자유롭게 모델을 활용.
- *   **완벽한 통제권:** 모델의 동작, 데이터 처리 방식 등을 완벽하게 제어할 수 있음.
- *   **문제점:** 다양한 인공지능 모델 중에서 프로젝트의 특성과 목표에 가장 적합한 모델을 선택하기 어려움.
- *   **해결/배운점:** 개발 목표에 따라 적합한 모델을 선택하는 것이 중요함.
- *   **모델별 권장 사항:**
- *   **수익을 목표로 하는 개발:** 클로드(Claude) 모델 사용을 권장.
- *   **가성비를 중시하는 경우:** 안티그레비티(Anti-gravity) 모델을 고려.

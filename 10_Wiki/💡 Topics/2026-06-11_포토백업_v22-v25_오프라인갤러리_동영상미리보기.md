### 세부 내용
- **sqlite3.Row 함정** — Python `sqlite3.Row`는 `.get()` 미지원 → `AttributeError` 발생 → `p.keys()`로 확인 또는 `dict(p)` 변환 필요
- **isServerOnline 캐시 문제** — 동기화 시점의 상태가 메모리에 캐시되어, 서버 끈 후에도 `true` 유지 → 로컬 파일 존재만 확인하는 방식이 유일하게 신뢰 가능
- **동기화 전체 교체 부작용** — `clearPhotos()` → 재삽입 시 앱 전용 필드(`localThumbnailPath`) 소실 → 삭제 전 앱 전용 필드 별도 백업 필수
- **저장 위치 설계** — 외장하드(원본) / 노트북(썸네일+미리보기 원본) / 핸드폰(복사본, 오프라인용)
- **버그 5건 발견 및 해결** — sync API 500에러, 갤러리 타임아웃, 로컬경로 소실, isServerOnline 캐시, 오프라인 삭제 불가

### 8.2 추천 컬럼 구조

```
| 날짜 | 카테고리 | 질문 | 답변 | 출처 | 상태 |
```

### 8.3 데이터 수집 소스 3가지

**A) 본인 글에서 추출**

- 카톡/슬랙 답장 → Q&A 쌍
- 블로그·SNS 글 → 주제별 질문
- 강의 자료 → 자주 받는 질문
- 메모/일기 → 시그니처 표현

**B) AI 자동 증강**

ChatGPT/Claude 프롬프트:

```
이 Q&A를 5가지 다른 질문 표현에 매칭되게 변형해줘.
- 짧은 버전
- 정중한 버전
- 반말 버전
- 상황 추가
- 헷갈리는 사람용

[원본 Q&A]
```

> 🚀 1개 → 6개로 부풀리기

**C) 실사용 피드백 (가장 강력)**

LM Studio로 본인 AI랑 대화하면서:

- 🔴 헛소리한 질문 → 정답 작성해서 추가
- 🔴 톤이 어색한 답변 → 본인 톤으로 다시 써서 추가
- 🔴 답 못한 질문 → 답 만들어서 추가

### 8.4 주기적 발전 루프

```
주 1회 30분 투자:
  월: 이번 주 받은 질문 모으기 (10-15개)
  화: AI 증강으로 30-50개로 부풀리기
  수: 데이터셋에 추가 + 카테고리 균형 점검
  목: 부족한 부분 메모
  금: LM Studio에서 실사용
─────────────────────────────────────
2주마다: 재학습 → 새 GGUF 배포
```

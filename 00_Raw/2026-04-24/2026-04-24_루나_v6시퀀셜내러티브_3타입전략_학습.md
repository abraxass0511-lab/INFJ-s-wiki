### 2. 기존 v4 영상 분석 — 구조적 문제점

| 항목 | v4 문제점 |
|------|----------|
| **시각** | 단일 이미지 1장이 20초간 고정 → 시청자가 3초 만에 이탈 |
| **스토리** | 내러티브 없음 — 비트만 깔리고 텍스트가 올라감 |
| **Hook** | 첫 1.5초 전용 훅 없음 → 스크롤 멈출 이유가 없음 |
| **저장 동기** | 감성만 있고 정보/유용성 없음 → 저장/공유 동기 부족 |
| **음악/이미지** | 이전 영상과 동일한 음악/이미지 재사용 → "또 이거야?" 반응 |
| **완청률** | 변화 없는 화면 20초 → 완청률 저하 → 알고리즘 노출 감소 |

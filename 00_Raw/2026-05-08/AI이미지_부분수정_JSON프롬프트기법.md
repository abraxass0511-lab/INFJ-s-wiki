## 📌 Brief Summary
AI를 활용하여 이미지의 특정 부분만 수정하기 어려운 문제에 대한 해결책을 제시한다. 이미지를 JSON 프롬프트로 상세하게 구조화하여 AI 모델에게 필요한 부분만 선택적으로 변경하도록 지시하는 방법을 통해 이미지 편집의 정밀도를 높일 수 있다.

## 📖 Core Content

### 문제 상황 / 배경
AI 이미지 생성 또는 편집 과정에서 전체 이미지가 아닌 특정 영역이나 객체만 수정하고 싶을 때, 해당 부분만을 정확하게 변경하는 것이 어렵다는 문제가 발생한다. 이는 AI 모델이 이미지 전체를 하나의 단위로 처리하려는 경향이 있어 부분적인 수정이 까다롭기 때문이다.

### 핵심 내용 / 해결 방법
이미지의 특정 부분만 효율적으로 수정하기 위한 해결책으로 'JSON 프롬프트' 기법이 제시된다. 이 방법은 다음과 같은 단계를 따른다:
1.  **이미지 상세화:** 수정하고자 하는 이미지를 JSON 프롬프트 형태로 상세하게 기술하여 AI 모델에게 이미지의 구성 요소와 각 부분의 특징을 명확하게 전달한다.
2.  **특정 부분 수정:** JSON 프롬프트 내에서 수정하고자 하는 특정 부분에 대한 정보만 변경하거나 추가하여 AI 모델이 해당 부분만을 집중적으로 수정하도록 유도한다.

이러한 접근 방식을 통해 AI 이미지 편집의 정밀도를 높이고 원하는 결과물을 얻을 수 있다.

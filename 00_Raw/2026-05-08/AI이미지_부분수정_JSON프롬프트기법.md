## 📌 Brief Summary
AI 생성 이미지에서 특정 부분만 수정하기 어려운 문제에 대한 해결책을 제시합니다. 이미지를 JSON 프롬프트를 활용하여 상세하게 분할한 후, 원하는 특정 부분만 효율적으로 수정하는 기법을 설명하며, 이는 AI 이미지 편집의 정밀도를 높이는 데 기여합니다.

## 📖 Core Content

### 문제 상황 / 배경
AI를 통해 생성된 이미지의 특정 부분만을 수정하고자 할 때, 해당 부분만 정확하게 변경하는 것이 어렵다는 문제가 발생합니다. 기존 방식으로는 전체 이미지를 다시 생성하거나 광범위하게 수정해야 하는 비효율성이 따를 수 있습니다.

### 핵심 내용 / 해결 방법
AI 이미지의 부분 수정 문제를 해결하기 위한 방법은 다음과 같습니다:

1.  **JSON 프롬프트 활용**: 이미지를 JSON 프롬프트 형태로 상세하게 구조화하고 분할합니다. 이는 이미지의 각 구성 요소를 개별적으로 인식하고 제어할 수 있게 합니다.
2.  **특정 부분 수정**: JSON 프롬프트로 상세하게 나뉜 이미지 구성 요소 중에서 수정하고자 하는 특정 부분만을 선택하여 변경합니다. 이 방법을 통해 전체 이미지에 영향을 주지 않고 원하는 부분만 정밀하게 수정할 수 있습니다.

# AI 이미지 부분 수정 JSON 프롬프트 기법

## 📌 Brief Summary
이 문서는 AI 이미지 생성 및 수정 과정에서 특정 부분만 변경하기 어려운 문제에 대한 해결책을 제시합니다. 이미지를 JSON 프롬프트로 상세하게 구조화하여 특정 영역만 정밀하게 수정할 수 있는 기법을 소개하며, 이를 통해 AI 이미지 편집의 효율성과 정확성을 높일 수 있습니다.

## 📖 Core Content

### 문제 상황 / 배경
AI를 활용하여 이미지를 생성하거나 기존 이미지를 수정할 때, 사용자가 이미지의 특정 부분만을 변경하고자 하는 경우가 빈번합니다. 하지만 일반적인 프롬프트로는 이미지 전체에 영향을 미치거나, 원하는 특정 부분만을 정확하게 수정하기 어려운 문제가 발생합니다. 이는 AI 모델이 이미지 전체를 하나의 맥락으로 인식하여 부분적인 변경이 어렵기 때문입니다.

### 핵심 내용 / 해결 방법
이미지의 특정 부분만 수정하기 어려운 문제를 해결하기 위해 'JSON 프롬프트' 기법을 활용할 수 있습니다.

1.  **이미지 상세화:** 수정하고자 하는 이미지를 JSON(JavaScript Object Notation) 형식의 프롬프트로 상세하게 나눕니다. 이 과정에서 이미지의 각 구성 요소, 객체, 배경, 스타일 등을 구조화된 데이터로 정의합니다.
2.  **특정 부분 지정 및 수정:** JSON 프롬프트 내에서 수정하고자 하는 특정 부분(예: 특정 객체의 색상, 위치, 형태 등)에 해당하는 필드만 변경합니다.
3.  **AI 모델 적용:** 수정된 JSON 프롬프트를 AI 이미지 생성 또는 편집 모델에 입력하여, 지정된 부분만 변경된 새로운 이미지를 얻습니다.

이 방법은 AI 모델에게 이미지의 구조와 각 부분의 역할을 명확하게 전달함으로써, 사용자가 원하는 특정 영역에 대한 정밀한 제어를 가능하게 합니다.

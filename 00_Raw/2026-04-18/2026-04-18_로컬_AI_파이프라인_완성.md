### 3. 렌더 엔진 v3 — BPM 오버라이드 패턴

```python
import render_enhanced as re_mod

# BPM 동적 변경
re_mod.BPM = 155
re_mod.BEAT_INTERVAL = 60.0 / 155
```
- `render_enhanced.py`를 직접 수정하지 않고 런타임 오버라이드

### 4. 이미지 → 함수 이름 충돌 주의

```python
# ❌ 잘못된 패턴 (이전 세션에서 발생)

render_enhanced(...)  # 함수? 모듈? → 충돌

# ✅ 올바른 패턴
```

# [[AI 이미지 부분 수정 — JSON 프롬프트 기법]]

## 📌 Brief Summary
AI 이미지 생성 시 전체를 재생성하지 않고, JSON 프롬프트로 이미지 구조를 상세하게 분해하여 특정 부분만 선택적으로 수정하는 기법을 학습함.

## 📖 Core Content

### 문제 상황
- 이미지에서 **일부분만 수정**하고 싶은데, 그 부분만 변경하기 어려움
- 전체 이미지가 재생성되어 원하지 않는 부분까지 변경되는 문제 발생

### 해결 방법
- 이미지를 **JSON 프롬프트로 상세하게 나눈 후**, 특정 부분만 수정하면 된다
- 이미지의 각 요소(배경, 인물, 오브젝트 등)를 JSON 구조로 분리하여 개별 제어 가능

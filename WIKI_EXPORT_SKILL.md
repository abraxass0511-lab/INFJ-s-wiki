# 🧠 P-Reinforce Wiki Export Skill

> 이 스킬은 모든 에이전트 워크스페이스에서 사용 가능합니다.
> 사용자가 "위키에 정리해줘", "오늘 배운 거 정리해줘" 등을 요청하면 이 스킬을 실행합니다.

## 트리거 문구
다음 중 하나라도 매칭되면 이 스킬을 실행:
- "위키에 정리해줘"
- "오늘 배운 거 정리"
- "지식 정리해줘"
- "wiki에 저장"
- "배운 거 저장"
- "학습 내용 정리"

## 실행 방법

현재 대화에서 학습한 핵심 내용을 추출하여, 아래 Node.js 명령을 실행합니다.

### 1단계: 학습 내용 구조화
대화를 분석하여 다음 항목을 추출합니다:
- **title**: 이 세션의 학습 주제 (한국어, 명확한 제목)
- **summary**: 한 줄 요약
- **keyLearnings**: 핵심 배운 점 (배열)
- **decisions**: 결정한 사항들 (배열)
- **skills**: 새로 익힌 스킬/패턴 (배열)
- **connections**: 관련 기존 지식 키워드 (배열)

### 2단계: 위키 에이전트의 chat-export 모듈 실행
```bash
node -e "import('C:/Users/YS/Desktop/안티그래피티/위키에이전트/src/chat-export.js').then(m => m.exportConversation({ title: '제목', summary: '요약', keyLearnings: ['항목1', '항목2'], decisions: ['결정1'], skills: ['스킬1'], connections: ['관련지식1'], conversationId: '현재_대화_ID' }))"
```

### 3단계: 결과 보고
실행 결과를 사용자에게 보고합니다:
- 어떤 카테고리에 배치되었는지
- 몇 개의 기존 지식과 연결되었는지
- RL 보상 점수

## 주의사항
- 반드시 실제 학습 내용만 추출. 잡담이나 디버깅 과정은 제외.
- 핵심 인사이트와 패턴을 중심으로 정리.
- connections에는 기존 위키 문서와 연결될 수 있는 키워드를 넣기.
